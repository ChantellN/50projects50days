# 50 Projects in 50 Days

A collection of 50 mini web projects built with HTML, CSS, and JavaScript.

## Projects

1. [Expanding Cards](./Expanding%20Cards) - Image panels that expand on click
2. [Progress Steps](./Progress%20Steps) - Animated progress steps component

## About

This repository contains projects from the "50 Projects in 50 Days" challenge, designed to improve HTML, CSS, and JavaScript skills through hands-on practice.

## Structure

Each project folder contains:
- `index.html` - HTML structure
- `style.css` - Styling
- `script.js` - JavaScript functionality

## Running Projects

Simply open the `index.html` file in any project folder with your web browser.
